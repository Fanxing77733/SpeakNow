## 第一步：启动 MySQL

**以管理员身份打开 PowerShell**，执行：

```powershell
net start MySQL
```

如果启动失败，尝试直接启动 mysqld：

```powershell
D:\mysql\mysql-8.0.40-winx64\bin\mysqld --console
```

MySQL 启动后，**另开一个终端**创建数据库：

```bash
D:\mysql\mysql-8.0.40-winx64\bin\mysql.exe -u root -p
```

输入密码后执行：

```sql
CREATE DATABASE IF NOT EXISTS english_speaking CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

> 注意：如果 root 密码不是 `root`，需要修改 `english-speaking-backend/es-server/src/main/resources/application.yml` 中 dev profile 的 `spring.datasource.password`。

---

## 第二步：安装并启动 Redis

### 方式一：使用 Windows 版 Redis（推荐）

1. 下载 Redis for Windows：https://github.com/tporadowski/redis/releases
2. 下载 `Redis-x64-5.0.14.1.msi` 或 `.zip`
3. 安装后 Redis 会自动启动（端口 6379）

### 方式二：使用 WSL（如果已安装 WSL）

```bash
wsl -d Ubuntu
sudo apt install redis-server
sudo service redis-server start
```

### 验证 Redis 是否启动

```bash
redis-cli ping
# 应返回 PONG
```

## 第三步：修改 application.yml 数据库密码（如需要）

文件：`h:\NLP-ES\english-speaking-backend\es-server\src\main\resources\application.yml`

找到 dev profile 部分：

```yaml
spring:
  config:
    activate:
      on-profile: dev
  datasource:
    username: root
    password: root       # ← 改成你的 MySQL root 密码
```

## 第四步：启动后端

```bash
cd h:\NLP-ES\english-speaking-backend
mvn spring-boot:run
```

首次启动时 Flyway 会自动执行 V1-V4 迁移脚本创建所有表并插入种子数据。

启动成功标志：
```
Started EsServerApplication in X.XXX seconds
```

后端运行在 `http://localhost:8080`。

---

## 第五步：启动前端

```bash
cd h:\NLP-ES\english-speaking-frontend
```

前端运行在 `http://localhost:5173`，API 请求自动代理到 `localhost:8080`。

## 常见问题

**Q: MySQL 启动报错 "1077"？**
A: 以管理员身份运行终端，执行 `net start MySQL`。

**Q: 数据库连接失败？**
A: 检查 MySQL 是否已启动、密码是否正确、数据库 `english_speaking` 是否已创建。

**Q: Redis 连接失败？**
A: 开发阶段如果 Redis 不是必选项，可以先注释掉 application.yml 中 Redis 相关配置。但登录的 IP 限流和锁定功能需要 Redis。

**Q: 8080 端口被占用？**
A: 修改 application.yml 中 `server.port`，同时修改 `english-speaking-frontend/vite.config.ts` 中的代理目标端口。
