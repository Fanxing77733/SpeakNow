# 模块三：AI 发音跟读评测 -- V1.0 测试用例

> 对应需求：PRD 3.4.1 子模块 3.1 | 验收标准 REQ-004 | 版本 V1.0
> 对应设计：HLD 2.2.3.3 发音评测接口 + 图2-8 发音评测活动图 + 图2-30 发音评测时序图
>
> 已实现后端 3 个 API：
> - GET /api/v1/eval/content?difficulty= -- 跟读内容列表
> - POST /api/v1/eval/pronounce -- multipart 上传录音+评测
> - GET /api/v1/eval/result/{recordId} -- 查看历史记录
>
> 已实现前端页面：PracticePage（练习页）、PracticeResultPage（结果页）

## 二、异常流程测试

| 场景ID | 场景描述 | 触发条件 | 预期错误处理 |
|--------|----------|----------|--------------|
| TC-PRACTICE-101 | 录音时长 <0.5s | 用户快速点击录音按钮后立即松手，录音时长 0.3s | 前端拦截，不发起 POST 请求；Toast 提示："未检测到有效语音，请重新朗读"（对应友好文案"录音<0.5s"）；无音频上传流量 |
| TC-PRACTICE-102 | 录音时长 >60s（前端自动截断） | 用户持续录音超过 60s | 前端自动停止录音并截断到 60s；Toast 提示："录音时长超过60秒，已自动截断"；前 60s 音频正常提交评测 |
| TC-PRACTICE-103 | 录音超时后端二次校验 | 前端未拦截的 >60s 录音到达后端 | 后端返回 400，message："录音时长超过60秒，已自动截断"；不写入 practice_records；前端展示友好提示 |
| TC-PRACTICE-104 | 上传音频 >5MB | 用户高质量格式录音（如 60s 无损 WAV） | 前端校验，若 >=5MB 则 Toast 提示："录音文件过大，请控制在60秒以内"；不发起上传请求 |
| TC-PRACTICE-105 | ASR 返回空文本 | 用户录音内容无意义（如纯噪音、咳嗽声）或发音极其不清 | 后端返回 422，message："未能识别语音内容，请确保发音清晰、环境安静后重试"；practice_records 表写入 status=asr_failed；前端展示友好提示 + "重新录音"按钮 |
| TC-PRACTICE-106 | 发音评测 API 超时 | AI Gateway 调用评测 API 超过 10s（正常超时）或 20s（弱网超时）仍未返回 | 后端返回 503，message："服务繁忙，请稍后重试"；practice_records 表写入 status=eval_failed 或 timeout；前端展示提示 + "重新录音"按钮 |
| TC-PRACTICE-107 | 评测 API 返回噪声无法评分 | 环境嘈杂/非英语发音/多人同时说话 | 后端返回 422，message："环境嘈杂或发音不清，无法准确评分，建议更换安静环境后重试"；practice_records 表写入 status=eval_failed；前端展示提示 |
| TC-PRACTICE-108 | 同用户 <1s 内重复提交（QPS 超限） | 用户快速连续点击/发送两次评测请求（间隔 <1s） | 后端返回 429，message："操作过于频繁，请稍后再试"；前端拦截层应禁用按钮 1s，展示 Toast |
| TC-PRACTICE-109 | 未登录访问评测 API | 用户未携带 JWT Token 或 Token 无效 | 后端返回 401，message："登录已过期，请重新登录"；前端捕获 401 → 保存当前路径到 sessionStorage → 跳转登录页 |
| TC-PRACTICE-110 | 麦克风权限被拒绝 | 用户在浏览器弹窗中选择"拒绝"麦克风权限 | 前端检测 navigator.mediaDevices.getUserMedia 抛出 NotAllowedError；展示友好提示："需要麦克风权限才能录音，请在浏览器设置中允许麦克风访问"；不进入录音状态 |
| TC-PRACTICE-111 | 麦克风硬件不可用 | 设备无麦克风或被其他应用占用 | 前端检测 NotFoundError 或 NotReadableError；展示友好提示："未检测到麦克风设备，请检查设备连接后重试" |
| TC-PRACTICE-112 | 网络中断时上传录音 | 录音完成后提交评测过程中网络断开 | 录音保留在 IndexedDB 中（upload_queue）；展示 Toast："网络不稳定，请重试"；弱网恢复后自动重试（interval 1s/2s/4s 指数退避共 3 次），成功后清除 IndexedDB 记录；3 次均失败则在下次进入页面时提示有未上传录音 |
| TC-PRACTICE-113 | 弱网连续超时 | 连续 3 次请求超时 >10s | 前端 Axios 拦截器判定为弱网；页面顶部展示黄色提示条："当前网络较慢，请耐心等待"；评测流程不中断，用户可继续操作 |
| TC-PRACTICE-114 | 后端 ffmpeg 转码失败 | 上传的音频格式为不支持的编码（如 FLAC 非标准编码） | 后端返回 422，message："音频格式不支持，请使用标准格式录音"（或通用兜底文案"未能识别语音内容"）；不写入 practice_records |
| TC-PRACTICE-115 | 查看不存在的评测记录 | GET /api/v1/eval/result/{recordId} 但 recordId 不存在或不属于当前用户 | 后端返回 404，message："记录不存在"；前端展示"未找到该评测记录" |
| TC-PRACTICE-116 | content_sentences 表为空 | 题库中无跟读内容 | GET /api/v1/eval/content 返回 200 空列表；前端展示空状态占位图 + 引导文案："暂无跟读内容，敬请期待" |

## 四、API 契约检查

| 检查项 | 接口路径 | 检查内容 | 预期 |
|--------|----------|----------|------|
