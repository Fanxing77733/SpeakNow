# 模块四：学习进度可视化 V1.0 测试用例

> 版本：V1.0 MVP
> 测试范围：学习进度汇总 API（1 个后端 API）+ 学习进度页（ProgressPage）+ 首页学习概览（HomePage）
> 对应需求：PRD 3.5.1 子模块 4.1 -- 学习进度可视化
> 对应设计：HLD 2.2.2.4 节 -- 图2-35 学习进度可视化时序图
> 生成日期：2026-06-27

**已实现后端 1 个 API：**

- `GET /api/v1/progress/summary` -- 返回 `{ empty: boolean, summary: { totalPractices, totalDurationSeconds, highestScore, totalDurationFormatted } | null }`

**已实现前端 2 个页面：**

- `ProgressPage` -- 3 张数字卡片（渐增动画 + 骨架屏 + 空状态）
- `HomePage` -- 欢迎区 + 学习概览（缩小版三卡片）+ 2x2 快捷入口
