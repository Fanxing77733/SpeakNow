## 二、V1.0 MVP — 核心闭环（15 天）

### 2.1 版本目标

交付「测→评→练→看」最小闭环，验证产品核心价值。
- 注册→首次评测转化率 >70%
- 首次对话完成率 >50%
- 发音评测与人工评分偏差 ≤1 档
- 单次会话时长 >8 分钟

### 2.2 功能范围

| 模块 | 功能 | 实现程度 |
| 1 | 用户注册与基础画像 | 邮箱/手机注册 + 年龄 + 学习目标三选一 |
| 2 | 英语水平测评 | 20 题固定题，初/中/高三档 |
| 4+5 | AI 发音评测 | 三维评分（准确度/流利度/完整度）+ 逐词颜色 + 音素纠错 |
| 6+7+8 | 智能语音对话 | 5 场景 × 5 轮角色对话 + 独立评分调用（语法/相关性/流利度） |
| 12 | 学习进度 | 3 数字卡片 + 2 折线图（练习次数趋势 + 分数趋势） |
| — | 基础设施 | AI Gateway、审计日志、语音 168h 删除、HTTPS、全局限流、Prompt 注入防护 |

**不包含**：游戏化、社区、推荐、路径规划、管理后台、实时 TTS、流式 ASR

## 三---

## 三、V2.0 增强版 — 游戏化 + 社区（15 天）

### 3.1 版本目标

从"工具"升级为"平台"，通过游戏化和社区建立用户留存和裂变增长飞轮。

**成功指标**：
- 次日留存率 >40%
- 7 日留存率 >25%
- 人均每日练习次数 >5 次
- 社区日均 UGC >100 条

### 3.2 功能范围

#### 新增模块

| 模块 | 功能 | 说明 |
| **10** | 游戏化闯关学习 | 语音闯关（主题关卡链）+ 单词 PK（随机匹配）+ 积分勋章系统 + 排行榜（好友/全站） |
| **11** | 学习社区 | 学习小组（创建/加入）+ 组内语音挑战 + 匿名互评录音（AI 辅助评分）+ 每日话题讨论 |
| **3** | 学习路径规划 | 3 条规则驱动预设路径（中考/四六级/日常），每日自动推送任务 + 学习日历打卡 |

#### 模块升级

| 模块 | V1.0 | V2.0 |
| 6+7+8 | 5 场景融合 | **60+ 场景** + 语法纠错/角色扮演拆分为独立入口 |
| 4+5 | 三维评测 | **五维评测**（准确度/重音/连读/语调/节奏） |
| 12 | 数字卡片 + 折线图 | **雷达图**（多维度能力分布）+ 词汇量统计 |
| 1 | 基础画像 | 完整多维画像（学习偏好/可用时间/薄弱项标签） |

#### 基础设施

- 流式 ASR（WebSocket 实时识别）
- CDN 音频加速

## 四---

## 四、V3.0 完整版 — 管理后台 + 高级功能（10 天）

### 4.1 版本目标

补齐 B 端能力，搭建管理后台基础框架，升级 C 端核心体验，形成完整商业闭环。

**成功指标**：
- 教师后台可完成班级创建→作业布置→学生点评完整流程
- 运营后台可查看核心运营指标（DAU/MAU/留存/付费转化）
- 智能客服自动解决率 >60%

### 4.2 功能范围

#### 管理后台

| 模块 | 功能 | 说明 |
| **14** | 教师端后台 | 班级管理（创建/添加学生）+ 学习报告（全班概览+个体详情）+ 作业布置与点评（文字/语音） |
| **15** | 运营管理后台 | 用户管理（查询/封禁）+ 内容审核队列（AI 预审+人工复审）+ 运营数据看板（DAU/MAU/留存率/付费转化率/渠道来源） |

#### C 端升级

| 功能 | 说明 |
| **16** | 智能客服（FAQ 静态页 + LLM 对话，支持文字提问） |
| **实时 TTS** | 对话中 AI 回复支持语音播报（流式 TTS），文本对话→语音对话 |
| **新题型** | 话题陈述（30s 准备 + 1-2min 陈述 + LLM 评估） |
| **第三方登录** | 微信一键登录 |
| **离线模式** | 下载离线包，无网络跟读练习，联网后同步 |
